# Innovartus SaaS Platform

A minimal SaaS application with CI/CD, auto-scaling, and monitoring.

## 🚀 Live Demo
[https://innovartus-saas.onrender.com](https://innovartus-saas.onrender.com)

## 📊 Results

| Metric | Value |
|--------|-------|
| Deployment Time | ~2 minutes |
| Downtime | < 30 seconds |
| Scaling Ease | Very Easy |

## 🛠️ Tech Stack
- **Backend**: Flask (Python)
- **Deployment**: Render
- **Version Control**: GitHub
- **CI/CD**: Automatic on push
